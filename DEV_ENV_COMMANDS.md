## Local dev environment commands

Project root: `~/PycharmProjects/BestBrain`

### 1. Create virtualenv (once)

```bash
cd ~/PycharmProjects/BestBrain
py -m venv .venv
```

### 2. Activate venv

**Git Bash / MINGW64**

```bash
cd ~/PycharmProjects/BestBrain
source .venv/Scripts/activate
```

**PowerShell**

```powershell
cd C:\Users\bestb\PycharmProjects\BestBrain
.venv\Scripts\Activate.ps1
```

**CMD**

```cmd
cd C:\Users\bestb\PycharmProjects\BestBrain
.venv\Scripts\activate.bat
```
